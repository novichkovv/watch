Общая Инфа

Url для API http://paperboy.pro/api/:api_method/,
где :api_method - метод API описанный здесь (как ни странно)).
Запросы могут передоваться через GET или POST в зависимости от метода.
Результат выполнения выдается в формате JSON.
Формат ошибок:
{
    error: true,
    error_code: код ошибки,
    error_text: текст ошибки
}

Авторизация

Авторизация проводится проверкой GET или POST параметра 'secret'.

Методы

GET feed

- передает подборку из 10 постов для пользоавтеля
Параметры
Параметр Обязятельный Пояснение
fb_id Да ID пользователя в Facebook
Ответ
id - ID у нас в системе
entry_id - ID поста в Feedly
stream_id - ID stream (категория или фид) в Feedly
thumbnail - заглавное изображение
title
summary
keywords
author
publish_date
source_url - Ссылка на первоначальный источник
create_date - Дата сохранение в нашу БД
